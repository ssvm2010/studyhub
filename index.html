Ch 1 — Introduction to Computers

Types, Generations, I/O, Memory

▼
1 Mark
1
ENIAC stands for ___
▼
Electronic Numerical Integrator And Computer. First general-purpose electronic digital computer (1945).
2
Which generation uses VLSI chips?
▼
4th Generation (1971–present)
3
RAM is a ___ memory.
▼
Volatile — data lost on power off.
4
1 KB = ___ bytes
▼
1024 bytes
5
Which device is both input and output?
▼
Touch screen
2 Mark
1
Differentiate RAM and ROM.
▼
RAM: Volatile, read/write, temporary.
ROM: Non-volatile, read only, permanent (BIOS).
2
What is Cache Memory?
▼
Small ultra-fast memory between CPU and RAM. Types: L1, L2, L3 cache.
3 Mark
1
Classify computers based on size.
▼
Micro: Desktop, Laptop.
Mini: PDP-11.
Mainframe: IBM z-series.
Super: PARAM, Cray.
5 Mark
1
Explain generations of computers.
▼
Gen1(1940-56):Vacuum tubes|ENIAC
Gen2(1956-63):Transistors|IBM7094
Gen3(1964-71):IC chips|IBM360
Gen4(1971-now):VLSI|PC,Laptops
Gen5(Future):ULSI|AI,NLP
⭐ Often 5 Mark
10 Mark
1
Explain basic organization of a computer.
▼
1. Input Unit: Keyboard, Mouse.
2. CPU: ALU + CU + Registers.
3. Memory: RAM (temp), ROM (perm), Secondary.
4. Output: Monitor, Printer.
Input → CPU ↔ Memory → Output

Binary, Octal, Hex, Complement

▼
1 Mark
1

▼

2

▼

2 Mark
1
Convert (45)₁₀ to Binary.
▼
45÷2=22R1|22÷2=11R0|11÷2=5R1
5÷2=2R1|2÷2=1R0|1÷2=0R1 → 101101

2's complement of 1010 = ___
▼
1's comp=0101; +1 = 0110
3 Mark
1
Explain 1's and 2's complement.
▼
1's: Flip all bits. 1010→0101
2's: 1's comp + 1. 0101+1=0110
Use: Negative numbers & subtraction.
5 Mark
1
Number systems with conversions.
▼
(45)₁₀: Binary=101101 | Octal=55 | Hex=2D
⭐ Always 5 Mark!
10 Mark
1
All number system conversions with examples.
▼
Dec→Bin(÷2): 13→1101 | Bin→Dec(×2ⁿ): 1101=13
Dec→Oct(÷8): 100→144₈ | Dec→Hex(÷16): 255→FF
Bin→Oct(3bits): 110101→65₈ | Bin→Hex(4bits):11010101→D5₁₆
1's comp: flip bits | 2's comp: 1's+1

Ch 3 — Boolean Algebra & Logic Gates

Gates, Truth tables, De Morgan

▼
1 Mark
1
Universal gate = ___
▼
NAND and NOR
2
A+A' = ___
▼
1
3
(A·B)' = ___
▼
A' + B' (De Morgan)
5 Mark
1
Explain all basic logic gates.
▼
AND: Y=A·B | OR: Y=A+B | NOT: Y=A' | NAND: Y=(A·B)' | NOR: Y=(A+B)' | XOR: Y=A⊕B
AND:11→1 | OR:01→1 | XOR:01→1,11→0
⭐ Learn all gates!
10 Mark
1
Boolean algebra laws and De Morgan's theorems.
▼
Identity:A+0=A|A·1=A | Null:A+1=1|A·0=0
Complement:A+A'=1|A·A'=0 | Idempotent:A+A=A
De Morgan:(A+B)'=A'B'|(AB)'=A'+B'
